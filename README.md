# Java

Repositório de exercícios e projetos desenvolvidos em Java.

### Diretórios

| Pasta | Conteúdo |
|---------|---------|
| `01.fundamentos` | Lógica de programação, estruturas de controle, vetores, matrizes e métodos |
| `02.poo` | Programação orientada a objetos |
| `03.estruturas-e-algoritmos` | Busca, ordenação e recursividade |
| `04.projetos` | Aplicações de console integrando múltiplos conceitos |

### Objetivo

Documentar a evolução dos estudos em Java por meio da prática contínua.
