# Projetos

Projetos desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas,
utilizando Java e conceitos de Programação Orientada a Objetos.

Os projetos utilizam console, menus simples e estruturas vistas em aula,
sem interface gráfica ou banco de dados.

- Biblioteca: cadastro, aluguel e devolução de livros
- Sistema Bancário: saldo, cartão e compras aprovadas/negadas
- Loja: produtos físicos e digitais, interface de pagamento e polimorfismo
- Navegador de Arquivos: listagem e recursividade
- Artefatos Místicos: inventário, soma de poderes e busca
