# Programação Orientada a Objetos

Exercícios desenvolvidos durante a disciplina de Programação Orientada a Objetos,
com foco nos principais conceitos da POO em Java.

## Conteúdos abordados
- Fundamentos
- Construtores
- Encapsulamento
- Herança
- Polimorfismo
- Interfaces
- Coleções
- Exceções
- Recursividade
